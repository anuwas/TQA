@Qualifier Annotation

Now if autowiring is using and in bean.xml file multiple beans are available with same type . Then we can give a clue to spring that which bean needs to be autowire using @Qualifier annotation.

So a @Qualifer is something we can mention in bean definition to say this bean actually a circle related bean.

To do so need to declare a qualifier tag in bean.xml under the bean

Circle.java

public class Circle implements Shape{

 

       private Point center;

      

       public Point getPoint() {

              return center;

       }

 

       @Autowired

       @Qualifier("circleRelated")

       public void setPoint(Point center) {

              this.center = center;

       }

 

       @Override

       public void draw() {

              System.out.println("Drawing Circle ");

              System.out.println("Center of the cirecle is X "+center.getPointX()+", Y "+center.getPointY());

             

       }

}
