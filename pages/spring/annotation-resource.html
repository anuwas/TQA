26.1 @Resource Annotation

 

It does standard dependency injection by Name.

So the annotation is @Resource (name=”point2”). Here I am using point2 will be injected. Now suppose I want to inject dependency into Circle class for point2 then the code will be

26.1.1 Circle.java

 

public class Circle implements Shape{

 

       private Point center;

      

       public Point getPoint() {

              return center;

       }

 

       @Resource(name="pint2")

       public void setPoint(Point center) {

              this.center = center;

       }

 

       @Override

       public void draw() {

              System.out.println("Drawing Circle ");

              System.out.println("Center of the cirecle is X "+center.getPointX()+", Y "+center.getPointY());

             

       }

 

}

Now suppose, I didn’t declare name=”point2”, Then It will look for the any bean name is available in bean.xml file same name as member variable. I mean “centre”. Here “centre” is the member variable in Circle class. So if name is not declared with @Requied annotation then it will looking for the bean named as member variable name.

26.1.5 Init() & Destroy()

 

We have seen the init method and destroy method, the same thing we can implement using the Annotation

I have two functions in Circle class and I want one method to be call while circle initializing and another method to be called while it destroying.

26.1.5.1 MainClass.java

public class MainClass {

       public static void main(String[] args) {

              AbstractApplicationContext context = new ClassPathXmlApplicationContext("bean.xml");

              context.registerShutdownHook();

              Shape shape = (Shape) context.getBean("circle");

              shape.draw();

             

       }

}

26.1.5.2 Circle.java

public class Circle implements Shape{

 

       private Point center;

      

       public Point getPoint() {

              return center;

       }

 

       @Resource(name="pint2")

       public void setPoint(Point center) {

              this.center = center;

       }

 

       @Override

       public void draw() {

              System.out.println("Drawing Circle ");

              System.out.println("Center of the cirecle is X "+center.getPointX()+", Y "+center.getPointY());

             

       }

      

       @PostConstruct

       public void initializeCircle() {

              System.out.println("Circle initializing .. ");

       }

      

       @PreDestroy

       public void destroyCircle() {

              System.out.println("Circle Destroing .. ");

       }
