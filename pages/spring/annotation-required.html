@Requied Annotation

@Required Annotation

If any setter method is being used and dependency is declared in bean.xml file , then when that piece of code executes spring will throw an null pointer exception. So null pointer exception is a generic type of exception and it throws in runtime, so it is difficult to get why null pointer exception coming.

To avoid this kind of situation @Requied annotation is helpful. If we declare @Requied annotation on any setter method and dependency is not declare in bean.xml file then when the application being up and when all the bean initialize spring will throw an exception for that particular dependency is not declare .

So the exception coming before the application start up.

A BeanPostProcessor (RequiredAnnotationBeanPostProcessor) needs to be declared in bean.xml file, which will validate the @Requied annotation.

Circle.java

public class Circle implements Shape{

 

       private Point center;

      

       public Point getPoint() {

              return center;

       }

 

       @Required

       public void setPoint(Point center) {

              this.center = center;

       }

 

       @Override

       public void draw() {

              System.out.println("Drawing Circle ");

              System.out.println("Center of the cirecle is X "+center.getPointX()+", Y "+center.getPointY());

             

       }

 

}

 

bean.xml

<?xml version="1.0" encoding="UTF-8"?>

<beans xmlns="http://www.springframework.org/schema/beans"

       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"

       xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd">

 

<bean id="triangle" class="com.anu.tqa.requiedannotation.Triangle">

       <property name="pointA" ref="pint0"/>

       <property name="pointB" ref="pint1"/>

       <property name="pointC" ref="pint2"/>

</bean>

 

<bean id="pint0" class="com.anu.tqa.requiedannotation.Point">

       <property name="pointX" value="0"/>

       <property name="pointY" value="0"/>

</bean>

 

<bean id="pint1" class="com.anu.tqa.requiedannotation.Point">

       <property name="pointX" value="0"/>

       <property name="pointY" value="20"/>

</bean>

 

<bean id="pint2" class="com.anu.tqa.requiedannotation.Point">

       <property name="pointX" value="20"/>

       <property name="pointY" value="0"/>

</bean>

 

<bean id="circle" class="com.anu.tqa.requiedannotation.Circle">

       <!-- <property name="point" ref="pint0"/> -->

</bean>

<bean class="org.springframework.beans.factory.annotation.RequiredAnnotationBeanPostProcessor"/>

</beans>
