Spring Container

Spring container is the container of bean. So the pojo that we have created can be containing in spring container. And spring mange all those object in the container. It handles the life cycle of the object, handles the instantiate of the object and destruction of the object. We can have object outside of the spring container, but facility to keep object inside the container is that spring mange the object by its own.
