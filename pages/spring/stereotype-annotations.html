Stereotype Annotations

Now <context:component-scan base-package=””></context> tag search for few other annotations apart from @Components. These annotations are acutally called Stereotype Annotations.

 

Stereotype Annotations

So the concept of Stereotype Annotation is something like this, when you write enterprise applications in spring you would have some standar beans that perform some standar rules. You would have a data object , you would have a service class you would have a view you would have a controller so all these are stereotypically roles that some spring beans perform. So what spring done here it has some standard annotations that let us define to perform those particular roles.
