</bean>

 

Another way of autowiring is “byTpe” , and it works only when there is only one member variable and only one bean of each type. It checks the class with the member variable class and ignore the name.

For this particular case autowiring will not work, because there are three point variable. So spring cannot assume what would be the type, as we know that we should have only one member variable required for autowiring by type.
