@Controller Annotation

 @Controller

If Circle is a controller then we can use @Controller

@Controller

public class Circle implements Shape{

 

       private Point center;

      

       public Point getPoint() {

              return center;

       }

 

       @Resource

       public void setPoint(Point center) {

              this.center = center;

       }

 

       @Override

       public void draw() {

              System.out.println("Drawing Circle ");

              System.out.println("Center of the cirecle is X "+center.getPointX()+", Y "+center.getPointY());

             

       }

}

So @Component is bean, but if we use these annotations then spring know what roles this class performing .One of the reason is that it helps us for level of documentation. Like it inform what roles this bean does.
